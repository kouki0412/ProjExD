# 第3回
## 迷路ゲーム：迷えるこうかとん
### ゲーム概要
- ex03/maze.pyを実行すると，1500x900にcanvasに迷路が描画され，迷路に沿ってこうかとんを移動させるゲーム
- 実行するたびに迷路の構造は変化する
### 操作方法
- 矢印キーでこうかとんを上下左右に移動する
### 追加機能
- スタート位置のランダム化とゴールの設定
- 自動的にゴールする機能(sキーを押すと動く)

### 追加したかったけどできなかったこと
- ハードモード(通った道が壁になって戻れない) (自動化がバグりまくって時間が無くなってしまったのでできなかった)
- 迷路が時間経過で変化する(通れるようになったりならなかったり)
- ゴール時のアクション
